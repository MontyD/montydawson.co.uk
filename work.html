My work

I will happily admit that I am just starting out and still learning, but I've found it far more enjoyable to learn through doing. So below are a few different learning exercises that I've put myself through. I've put a lot of myself into them and thoroughly enjoyed the work I've put into them.

The Well | Hopton

The idea was to create a simple website which contained the menu, opening hours, some photos, and a bit of information about the purpose that this Christian Coffee Shop serves. I took my time and did things from scratch, creating the CSS and JavaScript without any frameworks. Because the content was open to change and there was the change of adding more functionality later on, I decided to use the very flexible PHP CMS SilverStripe. This was the biggest challenge for me, and helped to develop my back-end skills.

I am very happy with the result, it draws on elements of The Well's branding and colours, it's fully responsive, the content is presented well, and the CMS is user-friendly so that anyone can make changes
